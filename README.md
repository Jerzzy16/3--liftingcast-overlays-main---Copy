## Getting started

```
Create .env

Copy .env.sample to .env

cp .env.sample .env


Fill in your api key = "NA", meet id, and meet password.

```
Install dependencies

```
npm install
```
Run development server with hot reloading.
```
npm run dev or start
```

Edit graphics as needed.
